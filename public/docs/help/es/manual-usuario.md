# Manual del Usuario (ES)

## Primeros pasos
1. Regístrate y confirma tu correo (revisa **Spam**).
2. Inicia sesión.
3. La **Organización** se crea automáticamente para el dueño de la cuenta.
4. Crea un **Proyecto** y agrega **Tareas**.

## Correo no confirmado
Si aparece **"Email not confirmed"**:
- Pulsa **Reenviar confirmación**
- Revisa **Spam**
