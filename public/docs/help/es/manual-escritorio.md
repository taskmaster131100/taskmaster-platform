# Manual Oficina / Multiartista (ES)

## Estructura
- **Organización** = productora/oficina.
- **Equipo** = miembros invitados.

## Invitar equipo
1. Ve a **Equipo**.
2. Pulsa **Invitar miembro**.
3. Copia el enlace y envíalo.
4. El usuario abre el enlace y pulsa **Aceptar invitación**.
