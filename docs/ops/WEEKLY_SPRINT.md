# Weekly Sprint (Marketing + Sales + Partnerships)

Objetivo: parar o modo “assunto solto” e transformar tudo em **plano semanal + execução diária**.

## Regras do jogo (simples)
1) **Uma fila única**: tudo entra no `MKT_SALES_TRACKER.csv` (não existe tarefa “no ar”).
2) **WIP limitado**: no máximo **3 tarefas em Doing** por vez.
3) **Toda ideia nova vira Inbox**: se surgir assunto novo no chat, o assistente checa o tracker:
   - se já existe → atualiza *Next step/Notes*.
   - se não existe → cria linha nova com Status=Backlog e Priority.
4) **Planejamento semanal 1x** + **check-in diário**.

### Agenda sugerida (cronograma)
**Segunda:** definir oferta + scripts + pipeline
**Terça:** rodar outreach (lote 1) + coletar objeções + ajustar copy
**Quarta:** outreach (lote 2) + follow-up 48–72h + fechar 1ª venda
**Quinta:** ajustar onboarding + templates + prova (prints)
**Sexta:** parceria (1 outreach) + relatório semanal + priorização próxima semana

> Ajuste conforme a meta da semana (vendas vs produto vs parcerias).

## Inbox / assuntos novos
Quando você falar algo novo (“preciso fazer X”), a pergunta é só uma:
**Isso é tarefa, decisão ou informação?**
- **Tarefa** → entra no tracker.
- **Decisão** → vira item no tracker como “Decidir X” (P0) com prazo.
- **Informação** → vai pra Notes da tarefa relevante.
