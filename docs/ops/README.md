# Ops (Marketing + Sales)

Arquivos:
- `MKT_SALES_TRACKER.csv` — lista única de tarefas (marketing/vendas/parcerias/ops)
- `WEEKLY_SPRINT.md` — planejamento semanal + regras (anti-perdido)
- `INBOX.md` — entrada rápida de assuntos novos (assistente move pro tracker)
- `DAILY_COMMAND.md` — rotina e direção do dia
- `DAILY_UPDATE_TEMPLATE.md` — template do seu update diário

Uso diário:
1) Marcos envia update usando o template.
2) Assistente responde com: meta do dia + 3 ações + riscos.
3) Atualizar status no tracker.
